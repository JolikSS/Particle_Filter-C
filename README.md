# Фильтр частиц (Particle Filter) на C

[![Language](https://img.shields.io/badge/язык-C-blue.svg)](https://ru.wikipedia.org/wiki/C)

Реализация метода фильтрации частиц (Particle Filter / Sequential Monte Carlo) для отслеживания движущегося объекта с шумными измерениями.

## 🎯 Что это такое?

Фильтр частиц — это метод отслеживания объекта, который использует множество гипотез (частиц) о положении объекта. Каждая частица — это предположение "где может быть объект". Чем ближе частица к реальному объекту — тем больше её вес.

**Особенности реализации:**
- ✅ 500 частиц для высокой точности
- ✅ Адаптивный шум (подстраивается под ошибку)
- ✅ Ресемплинг для борьбы с вырождением
- ✅ 95% доверительный интервал
- ✅ Экспорт результатов в CSV
